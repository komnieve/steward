# Transcription Corrections

Common speech-to-text errors and their corrections. Use this to auto-correct when transcription gets things wrong.

If you use voice input (dictation), speech-to-text will regularly garble proper nouns, technical terms, and names. Maintaining a correction table lets the runtime auto-fix these during conversations.

## How to use

1. Add entries below as you discover them
2. Reference this file in your runtime's session-instructions file (`CLAUDE.md`, `AGENTS.md`, etc.) session-start ritual
3. Also add a quick-reference table directly in that session-instructions file for the most common ones
