# Example Practice Layers

*This directory is a home for fully-written Practice Layers authored by users or
communities that have adapted Steward for their context.*

*Currently empty — the public repo ships only the blank templates in* `../templates/`.

If you've authored a Practice Layer you want to share — as a fork, a community edition,
or a personal example that might help someone else — open a PR adding a subdirectory
here with your version. Include a short `README.md` explaining who it's for and what
tradition or framing it draws from.

Examples of the kind of thing that would fit here (imagined, not yet written):

- A Stoic-flavored Practice Layer drawing on Marcus Aurelius and Epictetus
- A Christian contemplative layer drawing on the Examen and Lectio Divina
- A secular humanist layer grounded in values clarification and ACT
- A service-oriented layer for volunteers and community organizers
- A creator-oriented layer for writers, artists, and craftspeople

Gift economy: fork freely, share when it serves, no gatekeeping.
