# Wholesomeness lens

> *→ See [`../examples/contemplative-practitioner/wholesomeness-lens.md`](../examples/contemplative-practitioner/wholesomeness-lens.md) for a fetter-based worked example at depth.*

**Wholesome = doesn't feed the patterns you've named; ideally, loosens them.**

This is a negative test, not a positive one. Positive definitions of "wholesome" always
go vague. The negative test stays concrete because the patterns are finite, named, and
checkable.

Rewrite the list below to match your actual operative patterns. Traditional
contemplative traditions offer ready vocabulary (see below) but any precise name works.
What matters is that *you* recognize the pattern when it's running.

## How to use this lens

**This is a lens, not a checklist.** Always-on background, never ticked-off foreground.
The moment "did I run the wholesomeness check today?" becomes a ritual, the lens has
itself become the thing it's meant to catch — you're bargaining with the system for the
feeling of having-checked rather than doing the work. Name that irony precisely when it
starts.

Use in three moments:

1. **Before a meaningful decision** — which of my patterns is pulling on it?
2. **When stuck** — which pattern is running the stuckness? Name the specific one, not
   generic "avoidance."
3. **In reflection** — when patterns are visible across a day or week, diagnose
   precisely rather than generic self-talk.

**One pattern per observation.** Don't recite the list. Pick the one that actually fits
this moment.

## Reference vocabulary

If you want ready-made vocabulary, contemplative traditions offer specific terms. Take
what resonates, leave the rest.

- **Bargaining with reality** — the belief there's a way to strike deals with how
  things are. "If I do X, the universe owes me Y."
- **Conceit** — the felt need to be superior, equal, or inferior to someone.
  Over-preparation to avoid looking dumb; the inverse, shrinking, pre-apologizing.
  Either direction is conceit.
- **Restlessness** — can't sit with discomfort. Scatter, tab-hopping, the scroll-at-desk.
- **Identity view** — protecting a self-image from being tested. "I'm the kind of
  person who…" / "I shouldn't be seen as…"
- **Ill-will** — aversion toward another driving the action.
- **Sensual grasping** — reaching for pleasant, fleeing unpleasant. Often a subset of
  restlessness.
- **Doubt** — acting from distrust of the practice/path.

## How the steward uses this

The steward loads this file every run. Before it proposes a recommendation, it checks:
*does the framing still carry any of the patterns you named?* If yes, it keeps the
action but rewrites the framing. Action can be right; framing can still feed a pattern.
Fix the framing.

The steward never recites the list as content. It just uses it quietly.
