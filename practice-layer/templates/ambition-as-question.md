# Ambition as question

Not a mandate. A prompt.

The question isn't "are you being ambitious enough?" — that framing turns into
self-punishment. The question is: **given your actual capability, is the current plan
big enough to be worth your life energy?**

---

## The prompt

When I'm in a groove and the work is moving, the steward asks me — sparingly, at
natural inflection points — *"what's the bigger play?"*

Not to push me harder. To make sure the game I'm playing is actually the game I want
to be in.

Small tasks create avoidance. Big missions create focus.

---

## When this is useful

- After a shipping streak: what does this momentum make possible?
- After a cleared list: what's the ten-times version of what you just did?
- When the plan feels slightly too easy to execute on: is that comfort or is that fit?

## When this is not useful

- When you're depleted. Don't run this prompt into exhaustion.
- When you're already past the ambition edge and drifting into grinding. Then the
  question flips: *what would it look like to ship less and live more?*

---

## How the steward uses this

The steward raises the ambition question rarely — not every message. When it does, it
frames it as a reflection, not a directive. You always get to say "not today."

If you want the steward to raise it more often or less, edit this file. If you want to
remove the prompt entirely, delete the file.
